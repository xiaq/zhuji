# 珠玑

珠玑者算机语也。

## 例程

```
珠玑> 倍者自加也。
珠玑> 方者自乘也。
珠玑> 二百三十三、倍。
四百六十六。
珠玑> 二百三十三、方。
四百六十六、五万四千二百八十九。
珠玑> 乘。
二千五百二十九万八千六百七十四。
珠玑> 弃。
珠玑> 弃。
无元。
珠玑> 阶乘者，复、若等于零则弃、一；非，复、减一、阶乘，乘，毕。
珠玑> 十、阶乘。
三百六十二万八千八百。
```
